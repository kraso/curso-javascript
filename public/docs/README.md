# Curso de JavaScript en Español

## Acerca de este curso

Este es un curso completo de JavaScript en español, diseñado para llevar al estudiante desde los fundamentos hasta conceptos avanzados. El contenido está fusionado y adaptado de las mejores fuentes de aprendizaje disponibles:

| Fuente | Tipo | Licencia |
|--------|------|----------|
| [JavaScript.info](https://es.javascript.info) | Tutorial moderno | CC BY-NC-SA |
| [Eloquent JavaScript](https://eloquent-javascript-es.vercel.app/) | Libro en línea | CC BY-NC / MIT |
| [Microsoft Web Dev for Beginners](https://github.com/microsoft/Web-Dev-For-Beginners) | Curso práctico | MIT |

## Cómo usar este curso

1. **Empieza por el Nivel 1** si eres principiante
2. **Lee la teoría** de cada lección
3. **Practica los ejemplos** escribiendo el código tú mismo
4. **Resuelve los ejercicios** antes de ver las soluciones
5. **Pasa al examen** del nivel cuando te sientas cómodo
6. **Continúa al siguiente nivel** cuando apruebes (35+ puntos)
